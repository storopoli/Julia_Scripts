In this notebook we analyze Nealâ€™s Funnel (Neal, 2011). Neal (2011) defines a distribution that exemplifies the difficulties of sampling from some hierarchical models. Nealâ€™s example is fairly extreme, but can be trivially reparameterized in such a way as to make sampling straightforward. Nealâ€™s example has support for \(y \in \mathbb{R}\) and \(x \in \mathbb{R}^9\) with density

\[ p(y,x) = \text{Normal}(y \mid 0,3) \times \prod_{n=1}^9 \text{normal}\left(x_n \mid 0,\exp\left(\frac{y}{2}\right)\right). \]

The probability contours are shaped like ten-dimensional funnels. The funnelâ€™s neck is particularly sharp because of the exponential function applied to \(y\). So what if we reparameterize so that we can express \(y\) and \(x_n\) as standard normal distributions, by using a reparameterization trick2:

\[ \begin{aligned} x^* &\sim \text{Normal}(0, 0)\\ x &= x^* \cdot \sigma_x + \mu_x \end{aligned} \]

So, we can provide the MCMC sampler a better-behaved posterior geometry to explore: \[ \begin{aligned} p(y^*,x^*) &= \text{Normal}(y^* \mid 0,0) \times \prod_{n=1}^9 \text{Normal}(x^*_n \mid 0,0)\\ y &= 3y^*\\ x_n &= \exp \left( \frac{y}{2} \right) ) x^*_n. \end{aligned} \]
